- Lession 3: Mô hình MVC sử dụng lập trình hướng đối tượng, autoload
- Lession 4: Bổ sung routes, rewrite URL sử dụng htaccess 
- Lession 5: Model, sử dụng PDO tương tác với dữ liệu Mysql.
